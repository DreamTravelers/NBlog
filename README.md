# NBlog
SpringBoot + vue + Element-UI 搭建的博客系统

## 网站访问连接：
NBlog : [http://nblog.org.cn](http://nblog.org.cn)

## 网站技术

- **前端**： vue + vuex + vue-router + axios + element + echarts

- **后端**： SpringBoot + nginx + docker  + Swagger2 + MyBatis + Mysql + Redis + elasticsearch + RabbitMQ
